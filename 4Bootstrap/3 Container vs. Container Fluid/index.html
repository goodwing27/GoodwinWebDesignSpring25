Container vs. Container-Fluid

Container:

The ".container" class provides a responsive fixed width container. It has a set width no matter the size of the device and doesn't span the entire page. Everything is centered horizontally.

Container-Fluid:

The ".container-fluid" class provides a full width container, spanning the entire width of the viewport. It takes up the entire width of the page. This also means the size of the device doesn't matter and it can adapt to a new size if needed. Not always centered horizontally.
